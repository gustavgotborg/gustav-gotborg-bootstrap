ELEMENT

Element definierar vad som ska hända med innehållet inom start och sluttaggen.

HTML

Html är det språk (syntax) som är vanligast idag för att skapa webbsidor.

ID

Id ger html elementen ett specifikt id som tillåter dig att ändra saker för just det elementet.

CLASS

Class ger elementen en eller flera klasser för att lättare tilldela element en viss stylesheet kod

META

Meta taggen tilldelar metadata (information om data) som handlar om ett html dokument

Responsive

En responsiv hemsida är en hemsida som ändrar storlek och layout automatiskt för att se bra ut på alla olika enheter

HTTP

HTTP (HyperText Transfer Protocol) överför sidor från webb serverar till webb browsern

HTTPS

HTTPS fungerar som HTTP men krypterar innehållet på sidan

COMPRESSED DATA

Compressed data innebär att man komprimerar filerna genom komprimerad kodning om kortar ner koden till få rader

CDN

CDN (Content Delivery Network) är flera server utspridda på många ställen för att minska trafiken på hemsidan

LOCAL BOOTSTRAP

Den lokala kopian av bootstrap som du laddar ner och länkar till i ditt html dokument. Det ger dig en färdig css som redan är anpassad för alla enheter

JQUERY

Ett bibliotek av javascript funktioner för hemsidor som förenklar användningen av animationer och andra javascript för webbutveckklare

FLEXBOX

Flexbox används inom html för att på ett lätt sätt ordna föremål i rader och kolumner på hemsidan.

GRID

Grid används för att dela upp hemsidan i stora områden som sedan används för att göra det lättare när man ska bestämma positionering.

CODE SNIPPETS

Code snippets är delar av html kod som är lätta att återanvända i ditt egna projekt när du vill lägga till en speciell funktion. Istället för att skriva hela funktionens kod från början kan man istället återanvända färdiga delar av koden (Code snippets).
